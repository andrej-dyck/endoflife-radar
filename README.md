# End-of-Life Radar

An end-of-life radar for your projects using [endoflife.date](https://endoflife.date/).

## Development

Install dependencies with [pnpm](https://pnpm.io/)
```bash
pnpm i --frozen-lockfile
```

Run development environment
```bash
pnpm dev
```

Run preview build
```bash
pnpm preview
```

### Test & Lint

Run tests.
```bash
pnpm test
```

Run linter.
```bash
pnpm lint
```

### Production Build

```bash
pnpm build
```
